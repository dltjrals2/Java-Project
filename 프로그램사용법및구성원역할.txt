프로그램 사용법

소스코드가 들어있는 Termproject 폴더내의 
이미지가 들어있는 images폴더와
main문(Term클래스,gui클래스)가 들어있는 gui프로젝트 폴더가
같은 경로에 있어야 jar파일로 이미지파일을 로드하여 실행할수 있습니다.

1.서버 실행 Server.jar

2.서버 실행 확인후 클라이언트(포스) 두개 각각 실행 Client.jar
이때 login ID 치는순서대로 서버에서 0번, 1번 포스번호 설정

3.포스기를 이용하여 커피및 케이크 판매

판매 품목, 판매수량 결정하여 금액합산
좌측 커피버튼클릭 -> 커피판매 -> up버튼, down버튼으로 수량설정후 우측상단 홈버튼으로 홈복귀
좌측 케이크버튼 ->케이크판매 -> up버튼, down버튼으로 수량설정후 우측상단 홈버튼으로 홈복귀
수량 설정후 우측하단 동전버튼-> 결제 -> 결제액 및 구매완료 메세지 출력
이때 재고부족시 재고부족 메세지 출력

4.판매된 포스기의 번호(0번,1번)와 판매후 남은 재고수량 출력
이때 서버에서 현재 재고, 가격 정보 설정가능
클라이언트는 필수적으로 두개실행
서버에서
1.재고확인 2.재고수정 
셋중 입력을하여 필요한사항 수정
이때 맨처음 입력이 안될땐 한번더 엔터
1.재고확인 ->재고상황출력
2.재고수정 -> index
 ->	0:coffee개수 수정 
	1:cake개수 수정


처음 클라이언트 실행후 딜레이가 3초정도 있을수있습니다. 대기해주시면 해결됩니다.
서버 콘솔창에 마우스 커서가 있으면 클라이언트 실행이 정지될수있습니다.
